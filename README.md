# 🌐 AI Translator Web App

A simple and interactive web-based translator built using HTML, CSS, and JavaScript, powered by Microsoft Azure Translator API.

---

## 🚀 Features

* 🌍 Translate text into multiple languages
* 🔑 User-provided Azure API key and region
* 🎨 Modern UI with animations and colors
* ⏳ Loading animation and typing effect
* 💻 Responsive design for laptop screens

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Microsoft Azure Translator API

---

## 📂 Project Structure

/project-folder
│── index.html
│── style.css
│── script.js
│── README.md

---

## ⚙️ How to Run

1. Open the project folder
2. Double click `index.html`
3. Use your browser

---

## 🔑 How to Get Azure API Key

1. Go to Microsoft Azure Portal
2. Create a Translator resource
3. Copy API Key and Region

---

## ⚠️ Important Note

This project uses API key in frontend → not secure for production.
